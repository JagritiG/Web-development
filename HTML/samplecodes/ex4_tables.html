Creating Tables

Basic Table Structure: <table>, <tr>, <td>, <th>

The <table> element is used to create a table. The contents of the table are written out row by row.

The <tr> indicates the start of each row using the opening tag. (The tr stands for table row).
It is followed by one or more <td> elements (one for each cell in that row).
At the end of the row you use a closing <tr> tag.

The <td> represents each cell of a table.(The td stands for table data.)

The <th> is used just like the <td> element but its purpose is to represent the heading for either a column or a row.
(The th stands for table heading). For Example,

Country Population Male Female
United States 120M 70M 50M
India 220M 150M 70M
France 90M 40M 50M

Spanning Columns:

Sometimes you may need the entries in a table to stretch across more than one column.
The colspan attribute can be used on a <th> or <td> element and indicates how many columns that cell should run across.
For example,

9am 10am 11am 12am
Monday Geography Math Art
Tuesday Gym Home Ec

Spanning Rows:

Sometimes you may need the entries in a table to stretch across more than one row.
The rowspan attribute can be used on a <th> or <td> element and indicates how many columns that cell should run across.
For example,

ABC BBC CNN
6pm - 7pm Movie Comedy News
7pm - 8pm Sport Current Affairs

Long Tables: <thead>, <tbody>, <tfoot>

The headings of the table should sit inside the <thead> element.
The body should sit inside the <tbody> element.
The footer belongs inside the <tfoot> element.
For example,

Date Income Expenditure
1st January 250 36
2nd January 285 48
31st January 129 64
7824 1241

Width & Spacing:

The width attribute was used on the opening <table> tag to indicate how wide that table should be and on some opening <th> and <td> tags to specify the width of individual cells. The value of this attribute is the width of the table or cell in pixels.

The opening <table> tag could also use the cellpadding attribute to add space inside each cell of the table, and the cellspacing attribute to create space between each cell of the table. The values for these attributes were given in pixels.
For example,

Withdrawn Credit Balance
January 250.00 660.50 410.50
February 135.55 895.20 1170.15

Border & Background:

The border attribute was used on both the <th> and <td> elements to indicate the width of the border in pixels.

The bgcolor attribute was used to indicate background colors of either the entire table or individual table cells.
For example,

Withdrawn Credit Balance
January 250.00 660.50 410.50
February 135.55 895.20 1170.15
